# GitHub Pages Upload Instructions

1. Extract the ZIP locally.
2. Open the target GitHub repository.
3. Upload every extracted file directly to the repository root.
4. Replace files with matching names and commit to `main`.
5. Under **Settings → Pages**, use **Deploy from a branch**, branch `main`, folder `/ (root)`.
6. Wait for deployment, then open the Pages address in Safari.
7. Reload once, close the old Home Screen app completely and reopen it.

Do not upload the unopened ZIP as the website. `index.html` must be visible at the repository root.
