# Branding Implementation — v3.0.1

This release uses the approved Animus Companion Man-o’-War emblem for:

- Home Screen app icons at 192×192 and 512×512
- Apple Touch Icon at 180×180
- favicon assets at 16×16, 32×32, and 64×64
- the initial loading/synchronization screen
- the pre-existing compact header branding placement

The version 3.0 application remains the functional baseline. POI icons, category icons, map markers, route numbers, calibrated positions, map imagery, database content, storage keys, saved-user-data compatibility, navigation, gestures, routing, filters, progress tracking, favorites, notes, Captain’s Log, Jackdaw Planner, and all application functionality were intentionally left unchanged.

The service-worker cache name was incremented and all new branding assets were added to the offline cache. Activation continues to remove obsolete caches.
