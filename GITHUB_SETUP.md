# GitHub Pages Setup

This package is arranged for a brand-new GitHub repository. Upload the **contents of this folder** directly to the repository root.

## Recommended repository name

`AC-BlackFlag-Animus-Companion`

## Publish the app

1. Create a new public GitHub repository.
2. Choose **Add file → Upload files**.
3. Upload every extracted file, including `.nojekyll`.
4. Commit the files to the `main` branch.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select `main` and `/ (root)`, then save.
8. Wait for GitHub Pages to publish the site.
9. Open the published address in Safari and use **Share → Add to Home Screen**.

The ZIP itself should not be uploaded as the website. Extract it first.
